## Requirements

Minimum:
- pfSense >= 2.8.x
- CPU cores = 2
- RAM = 6 GB
- Storage = 12 GB
- Ethernet NICs = 4

Recommended:
- pfSense >= 2.8.x
- CPU cores >= 4
- RAM >= 12 GB
- Storage >= 32 GB
- Ethernet NICs >= 4
